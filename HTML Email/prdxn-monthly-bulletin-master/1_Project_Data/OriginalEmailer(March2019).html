Challenges Faced
Unniti Full VoIP - We used a third party VoIP tool (Voxter). We faced issues in the integration of some of the key features as it was completely a new tool for us and some of the features were not implemented from Voxter's end. Though this caused delay, we successfully completed this project with the help of Voxter team.
Shutterstock - We started with a test job with no knowledge of Bannerflow and After Effects, which was a challenge for us. But the team mastered both the tools and won the project.
We do the following for Shutterstock:
Review the AFX files client shares.
Work on AFX for6 sizes and 8+ languages.
Use the videos, created using AFX on Bannerflow to add body copy, CTA, etc.
